# Proyek Analisis Data: E-Commerce Public Dataset

## Deskripsi
Dashboard ini dikembangkan untuk menganalisis performa bisnis E-Commerce di Brazil, dengan fokus pada tren pendapatan, segmentasi pelanggan (RFM), dan distribusi geografis pelanggan.

## Struktur Direktori
- `/dashboard`: Berisi file utama untuk dashboard Streamlit (`dashboard.py`) dan data yang sudah dibersihkan (`main_data.csv`).
- `/data`: Berisi dataset asli dalam format CSV.
- `notebook.ipynb`: File analisis data lengkap mulai dari Wrangling hingga Visualisasi.
